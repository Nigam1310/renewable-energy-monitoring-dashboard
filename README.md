# Renewable Energy Monitoring Dashboard

A full-stack dashboard for monitoring renewable energy capacity across Indian states.

## Tech Stack

* React (Vite)
* Node.js
* Express
* Recharts
* Data.gov.in API

## Features

* State-wise renewable energy capacity
* Energy production visualization
* Renewable source distribution
* State-level analysis dashboard

## Run Locally

### Backend

```
cd backend
npm install
node server.js
```

### Frontend

```
cd frontend/renewable-dashboard
npm install
npm run dev
```

Backend runs on:
http://localhost:4000

Frontend runs on:
http://localhost:5173
